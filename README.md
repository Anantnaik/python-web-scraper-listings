# Python Web Scraper for Listing Websites

This project demonstrates a reusable Python web scraper built using BeautifulSoup and Pandas.

It extracts structured data from a listing-based website and exports clean CSV and Excel files.

## Features
- Scrapes product listing data
- Handles pagination
- Exports CSV and Excel formats
- Clean and structured output

## Tech Stack
- Python
- Requests
- BeautifulSoup
- Pandas

This scraper can be easily adapted for other websites such as job listings, directories, or e-commerce products.
